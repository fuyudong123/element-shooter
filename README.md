# 🎯 射击训练场

一款基于 Three.js 的 3D 第一人称射击游戏，**CS2 工业风场景 + 精致二次元角色**的混搭风格。浏览器直接运行，无需安装。

## 🎯 游戏目标

消灭竞技场中所有敌人即可获胜！地图上会随机刷新**血包**（❤️），注意拾取补充生命值。

## 🎮 操作方式

### 电脑端（键盘 + 鼠标）

| 操作 | 按键 |
|------|------|
| 移动 | **W / A / S / D** |
| 旋转视角 | **鼠标** |
| 射击 | **鼠标左键** |
| 跳跃 | **空格键** |
| 下蹲 | **Shift 键** |

### 手机端（触屏）

| 操作 | 方式 |
|------|------|
| 移动 | **左下半圆摇杆** 拖拽 |
| 旋转视角 | **右半屏幕** 滑动 |
| 射击 | 点击右下角 **⚡ 按钮** |
| 跳跃 | 点击左下角 **↑ 按钮** |
| 下蹲 | 点击右下角 **↓ 按钮** |

## ✨ 游戏特性

- 🏭 **CS2 工业风场景**：混凝土质感地面、金属立柱、油墨色墙壁，配合 ACES 色调映射营造射击场氛围
- 🎀 **精致二次元敌人**：大眼睛高光、飘逸头发、修身制服 + 领带，与写实场景形成鲜明反差
- 🤖 **智能敌人 AI**：巡逻 → 追击 → 绕圈射击 → 返回，自动绕开障碍物，卡住时自主逃脱
- 🔫 **写实武器模型**：金属质感手枪，带枪管、扳机护圈和木纹握把
- 💥 **命中反馈**：粒子爆炸、准星闪烁、相机震动
- ❤️ **血包系统**：地图随机刷新绿色十字血包，每12秒补充，最多同时存在3个
- 📱 **全平台适配**：电脑 + iPhone / iPad 触屏均可游玩，PWA 支持添加到桌面
- 📊 **数据统计**：游戏时长、射击数、命中率

## 🚀 快速开始

直接在浏览器打开：

**https://fuyudong123.github.io/element-shooter/**

## 🛠️ 本地运行

```bash
git clone https://github.com/fuyudong123/element-shooter.git
cd element-shooter
npx serve .
# 或 python -m http.server 8080
```

浏览器打开 `http://localhost:3000` 即可。

## 📂 技术栈

- **Three.js** — 3D 渲染引擎
- **CSS2DRenderer** — 血条 UI
- **MeshToonMaterial** — 二次元角色赛璐璐着色
- **MeshStandardMaterial** — CS2 风格场景物理材质
- **ACESFilmicToneMapping** — 电影级色调映射
- **原生 JavaScript** — 零框架依赖
- **GitHub Pages** — 静态托管部署

## 📱 添加到桌面（PWA）

1. iPhone **Safari** 打开游戏链接
2. 底部 **「分享」按钮** → **「添加到主屏幕」**
3. 桌面图标点击即玩
