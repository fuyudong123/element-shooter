# ✨ 元素射手

一款基于 Three.js 的 3D 第一人称射击游戏，灵感来自二次元风格。在浏览器中直接运行，无需安装任何软件。

## 🎯 游戏目标

消灭竞技场中所有带有元素属性的敌人即可获胜！

## 🎮 操作方式

### 电脑端（键盘 + 鼠标）

| 操作 | 按键 |
|------|------|
| 移动 | **W / A / S / D** |
| 旋转视角 | **鼠标** |
| 射击 | **鼠标左键** |
| 跳跃 | **空格键** |
| 下蹲 | **Shift 键** |

### 手机端（触屏）

| 操作 | 方式 |
|------|------|
| 移动 | **左下半圆摇杆** 拖拽 |
| 旋转视角 | **右半屏幕** 滑动 |
| 射击 | 点击右下角 **⚡ 按钮** |
| 跳跃 | 点击左下角 **↑ 按钮** |
| 下蹲 | 点击右下角 **↓ 按钮** |

## ✨ 游戏特性

- 🖌️ **二次元画风**：使用赛璐璐（Toon）材质，6种元素风格的Q版敌人（火/水/雷/风/冰/岩）
- 🤖 **智能敌人AI**：敌人会巡逻、追击、绕你走位射击
- 🔫 **能量武器**：带旋转能量核心和枪口光环特效
- 📱 **全平台适配**：电脑浏览器 + iPhone / iPad 触屏均可游玩
- 🌐 **PWA 支持**：iPhone Safari 可添加到主屏幕，体验近似原生 App
- 💥 **元素特效**：命中粒子爆炸、受击闪烁、击杀提示飘字
- 📊 **数据统计**：游戏时长、射击数、命中率

## 🚀 快速开始

直接在浏览器打开即可游玩：

**https://fuyudong123.github.io/element-shooter/**

无需下载、安装或注册。

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://github.com/fuyudong123/element-shooter.git

# 进入目录
cd element-shooter

# 使用任意 HTTP 服务器启动（任选其一）
npx serve .
# 或
python -m http.server 8080
# 或
npx http-server .
```

然后用浏览器打开 `http://localhost:3000`（端口取决于你用的工具）。

## 📂 技术栈

- **Three.js** — 3D 渲染引擎
- **CSS2DRenderer** — 血条 UI 渲染
- **MeshToonMaterial** — 卡通赛璐璐着色
- **原生 JavaScript** — 零框架依赖
- **GitHub Pages** — 静态托管部署

## 📱 添加到手机桌面（PWA）

1. 在 **iPhone Safari** 中打开游戏链接
2. 点击底部菜单的 **「分享」按钮**（方框箭头图标）
3. 向下滑动选择 **「添加到主屏幕」**
4. 点击右上角 **「添加」**
5. 桌面出现「元素射手」图标，点击即可游玩
